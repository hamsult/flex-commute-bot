# 출퇴근 봇 메시지 커스터마이징 가이드

봇에게 **DM**으로 명령어를 보내면 나만의 출퇴근 메시지를 설정할 수 있습니다.

---

## 기본 메시지 형식

커스터마이징을 하지 않으면 아래 기본 형식으로 발송됩니다.

```
홍길동님 코어출근, 09:03
```

---

## 변경 가능한 것 / 불가능한 것

| 항목 | 변경 가능 여부 | 설명 |
|------|:---:|------|
| 메시지 텍스트 | ✅ | 원하는 문구로 자유롭게 작성 |
| 이모지 | ✅ | 메시지 어디에든 삽입 가능 |
| **이름** | ❌ | Flex에 등록된 이름이 자동으로 들어감 |
| **출퇴근 시간** | ❌ | Flex에서 기록된 실제 시간이 자동으로 들어감 |
| **근무 속성** | ❌ | 코어, 자율, 재택 등 Flex에서 설정된 값 |

> 이름, 시간, 근무 속성은 Flex 데이터를 그대로 가져오므로 봇에서 임의로 바꿀 수 없습니다.

---

## 사용 가능한 변수

메시지 작성 시 아래 변수를 넣으면 발송 시 자동으로 실제 값으로 치환됩니다.

| 변수 | 치환되는 값 | 예시 |
|------|------|------|
| `{name}` | Flex 등록 이름 | `홍길동` |
| `{time}` | 출퇴근 기록 시간 (HH:MM) | `09:03` |
| `{status}` | Flex 근무 상태 전체 | `코어출근`, `자율퇴근`, `재택출근` 등 |

> 변수는 선택 사항입니다. 원하지 않으면 빼도 됩니다.

---

## 설정 가능한 상태 종류

| 상태 키워드 | 적용 시점 |
|------|------|
| `출근` | 코어출근, 자율출근, 재택출근 등 출근 시 |
| `퇴근` | 코어퇴근, 자율퇴근, 재택퇴근 등 퇴근 시 |
| `휴게시작` | 휴게 시작 시 (점심시간 12~13시는 알림 생략) |
| `휴게종료` | 휴게 종료 후 복귀 시 (점심시간 12~13시는 알림 생략) |

---

## 명령어

### 설정 — `set <상태> <메시지>`

```
set 출근 {name}님이 오늘도 힘차게 출근! 🎉 ({time})
set 퇴근 {name} 퇴근합니다 🌙 ({time})
set 휴게시작 ☕ {name} 잠깐 쉬어요
set 휴게종료 ⌨️ {name} 복귀했어요! ({time})
```

**예시 결과:**
```
홍길동님이 오늘도 힘차게 출근! 🎉 (09:03)
```

---

### 조회 — `list`

내가 설정한 커스텀 메시지 전체 확인

```
list
```

---

### 미리보기 — `preview <상태>`

실제 발송될 메시지 미리 확인 (현재 시간 기준)

```
preview 출근
preview 퇴근
```

---

### 초기화 — `clear <상태>` / `clearall`

```
clear 출근       → 출근 메시지만 기본값으로 초기화
clearall         → 전체 기본값으로 초기화
```

---

### 도움말 — `help`

```
help
```
